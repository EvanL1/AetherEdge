### `models_instances` (read-only)

|---|---|---|---|

### `models_instances_action` (**WRITE**)

Execute a control action on an instance. This writes to a real device via SHM + io.
The MCP host must provide `AETHER_ACCESS_TOKEN` containing a current signed
Admin or Engineer access token; local process access alone is insufficient.

| Parameter | Type | Required | Description |
|---|---|---|---|
| `instance_id` | integer | yes | Instance ID |
| `point_id` | string | yes | Numeric action point ID encoded as a string (for example, "1") |
| `value` | number | yes | Value to write |

### `models_instances_measurement` (**WRITE**)

Set a measurement value on an instance. This overwrites the live inst:{id}:M value -- the same field real device telemetry populates -- so rules, alarms, and dashboards will treat it as genuine until the next real update arrives.

| Parameter | Type | Required | Description |
|---|---|---|---|
| `instance_id` | integer | yes | Instance ID |
| `point_id` | string | yes | Point ID: numeric ("1") or semantic ("power_setpoint") |
| `value` | number | yes | Measurement value to set |

### `models_products` (read-only)

List available product types
