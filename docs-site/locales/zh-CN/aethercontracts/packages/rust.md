# AetherContracts Rust 基础库

> 权威来源：[AetherContracts](https://github.com/EvanL1/AetherContracts/blob/main/packages/rust/README.md)。此页面镜像到统一的 AetherIoT 文档中。

AetherContracts 的实验性 Rust 基础绑定。目前，它提供规范的 `u64` 解析，不会出现与语言无关的故障，并执行公共 CloudLink 固定清单。它不是完整的生产 Thing Model、身份验证或传输编解码器。
