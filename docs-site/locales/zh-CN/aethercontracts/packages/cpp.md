# AetherContracts C++ 基础库

> 权威来源：[AetherContracts](https://github.com/EvanL1/AetherContracts/blob/main/packages/cpp/README.md)。此页面镜像到统一的 AetherIoT 文档中。

C++17 目标 `AetherContracts::cpp` 是 C99 核心之上的轻量非持有包装层。它不会实现第二套解析器或协议语义。完整 JSON 支持和生产级 CloudLink 符合性仍在规划中；当前实验性夹具接口由 C99 核心提供。
