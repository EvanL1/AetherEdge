# AetherContracts TypeScript 基础库

> 权威来源：[AetherContracts](https://github.com/EvanL1/AetherContracts/blob/main/packages/typescript/README.md)。此页面已镜像到统一的 AetherIoT 文档中。

AetherContracts 的实验性 TypeScript 基础绑定。它目前提供规范 `uint64` 解析和与 RFC 8785 兼容的 JSON 规范化，以及一个实验性验证器，该验证器使用稳定的失败字符串执行公共 CloudLink 固定清单中的每个条目。它不是完整的生产 Thing Model、身份验证或传输编解码器。规范性契约保留在仓库发布包中。
